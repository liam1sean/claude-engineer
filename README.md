# claude-engineer (Learning Sandbox)

Beginner sandbox project for learning how to call Claude from Node.js.

## What this does
- Runs a small Node.js script (`index.js`)
- Sends a prompt to Claude
- Prints the response in the terminal
- Supports `.env` for your API key

## Prereqs
- Node.js installed
- An Anthropic API key (stored in `.env`, never committed)
